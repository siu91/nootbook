### 文档目录结构

**XXX部署 (存放与XXX部署相关文件 )**

​	**|__XXX部署文档 （XXX部署文档描述）** 

​    **|__config （相关配置文件）**

​    **|__bin （相关安装脚本）**



### 部署端规约

**{module} 指代安装模块名称** 

**{base}指代安装包基础目录，默认/deploy/install**

| 规约项             | 规约默认值              | 备注             |
| ------------------ | ----------------------- | ---------------- |
| 离线安装包存放目录 | {base}/{module}/package |                  |
| 模块配置文件       | {base}/{module}/config  |                  |
| 安装脚本           | {base}/{module}/bin     | 命名为install.sh |
| 安装目录           | /deploy/{module}        | 无特殊要求下     |
